NOTES

stopPropagation:

Purpose: Prevents the event from bubbling up to parent elements, but it does not prevent other event listeners on the same element from running.

Effect: The event will not trigger any further listeners on ancestor elements but will continue to execute other listeners attached to the same element.

stopImmediatePropagation:

Purpose: Stops the event from bubbling up to parent elements and also prevents any other event listeners on the same element from running.

Effect: Effect: The event will not bubble up, and no further listeners on the current element will be triggered.
